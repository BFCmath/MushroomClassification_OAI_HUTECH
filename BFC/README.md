# How to run

To reproduce the results or run the model, follow these steps:

1. **Prepare the Data**  
   - Place the dataset in the `data/` folder with the following structure:

      ```css
      data/
       train/
       test/
      ```

   - Ensure the dataset corresponds to the train and test subfolders.

2. **Install Dependencies**  
   Install the required Python packages using `requirements.txt`:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Model**  
   Execute the main script to train and evaluate the model:

   ```bash
   python main.py
   ```

4. **View Results**  
   The results will be saved in the `output/` folder.

5. **Refer to the Report**  
   For detailed methodology and analysis, refer to the [BFC_report.pdf](BFC_report.pdf) file included in this repository.
